This is a repository for some miscelaneous personal projects. The Python folder contains solutions to Project Euler problems. These problems contain mathematical/computational challenges where the difficulty lies in optimizing processing time complexity. Each solution should take less than a minute in processing time in order to compute the solution, meaning they must be very efficient. For more details, you can find the problem definitions on this page https://projecteuler.net/archives.
